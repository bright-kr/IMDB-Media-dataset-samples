# IMDB Media Dataset Sample

<h2>1,021개 레코드의 샘플 dataset</h2>

<a href="https://brightdata.co.kr">
    <img src="https://github.com/bright-kr/IMDB-Media-dataset-samples/blob/main/datasets-image.png" alt="Bright Data datasets" />

이 **IMDB Media Dataset Sample**에는 **1,021개 레코드**가 포함되어 있으며, **Bright Data API**를 사용해 추출되었습니다.

## 데이터 포인트

| 필드 이름 | 설명 |
|------------|-------------|
| `title` | 미디어 콘텐츠의 공식 제목입니다. |
| `popularity` | IMDB 플랫폼에서의 콘텐츠 인기도를 나타내는 지표입니다. |
| `genres` | 콘텐츠와 관련된 장르 목록입니다. |
| `presentation` | 미디어 콘텐츠에 대한 간단한 요약 또는 소개입니다. |
| `credit` | 감독 및 작가를 포함한 제작자 정보입니다. |
| `videos` | 미디어와 관련된 비디오 콘텐츠의 링크 및 세부 정보입니다. |
| `photos` | 미디어와 관련된 사진 콘텐츠의 링크 및 세부 정보입니다. |
| `top_cast` | 주요 출연 배우에 대한 정보입니다. |
| `details_release_date` | 미디어 콘텐츠의 공식 공개일입니다. |
| `details_countries_of_origin` | 미디어가 제작된 국가입니다. |
| `details_official_site` | 미디어 콘텐츠의 공식 웹사이트 링크입니다. |
| `details_language` | 미디어 콘텐츠의 주요 언어입니다. |
| `details_also_known_as` | 미디어 콘텐츠의 대체 제목 또는 다른 이름입니다. |
| `details_filming_locations` | 미디어가 촬영된 장소입니다. |
| `details_production_companies` | 미디어 제작에 참여한 회사들입니다. |
| `specs_color` | 미디어의 색상 사양에 대한 정보입니다. |
| `specs_sound_mix` | 미디어의 사운드 믹스 사양에 대한 세부 정보입니다. |
| `specs_aspect_ratio` | 미디어의 화면비 사양입니다. |
| `url` | 미디어의 IMDB 페이지 URL 링크입니다. |
| `comment` | 미디어에 대한 사용자 댓글 또는 리뷰입니다. |
| `media_type` | 영화 또는 TV 쇼와 같은 미디어 콘텐츠 유형입니다. |
| `imdb_rating` | IMDB에서의 미디어 평균 평점입니다. |
| `poster_url` | 미디어의 공식 포스터 이미지 URL입니다. |
| `imdb_rating_count` | IMDB에서 미디어가 받은 평점 수입니다. |
| `awards` | 미디어가 받은 수상 및 후보 지명 내역입니다. |
| `critics_review_count` | 미디어에 대해 제공되는 평론가 리뷰 수입니다. |
| `review_count` | 미디어의 전체 사용자 리뷰 수입니다. |
| `review_rating` | 사용자 리뷰의 평균 평점입니다. |
| `featured_review` | 미디어에 대한 주요 또는 주목할 만한 리뷰입니다. |
| `storyline` | 미디어의 자세한 줄거리 또는 스토리 설명입니다. |
| `boxoffice_budget` | 미디어 콘텐츠의 제작 예산입니다. |
| `id` | dataset 내 미디어 콘텐츠의 고유 식별자입니다. |


## 활용 사례

### 1. 영화 산업 분석
영화 장르와 시간에 따른 인기도 트렌드를 분석하여 제작 전략과 마케팅 활동에 활용할 수 있습니다.

### 2. 시장 조사
이 dataset을 사용해 미디어 소비에 대한 소비자 선호도와 행동을 연구함으로써 타겟 광고에 도움을 줄 수 있습니다.

### 3. 학술 연구
영화 제작, 공개 세부 정보, 관객 반응에 대한 포괄적인 데이터를 제공하여 미디어 연구를 지원합니다.



## 파일 형식

이 dataset은 여러 형식으로 제공됩니다:
- **CSV** (이 샘플에 포함됨)
- **JSON**
- **NDJSON**
- **JSON Lines**
- **Parquet**
- **Compressed (.gz)**
