이 **IMDB Media Dataset Sample**에는 **1,021개 레코드**가 포함되어 있으며 **Bright Data API**를 사용하여 추출되었습니다.

## Data Points

| Field Name | Description |
|------------|-------------|
| `title` | 미디어 콘텐츠의 공식 제목입니다. |
| `popularity` | IMDB 플랫폼에서 콘텐츠의 인기도를 측정한 값입니다. |
| `genres` | 콘텐츠와 관련된 장르 목록입니다. |
| `presentation` | 미디어 콘텐츠에 대한 간략한 요약 또는 소개입니다. |
| `credit` | 감독 및 작가를 포함한 제작진에 대한 상세 정보입니다. |
| `videos` | 미디어와 관련된 동영상 콘텐츠의 링크 및 상세 정보입니다. |
| `photos` | 미디어와 관련된 사진 콘텐츠의 링크 및 상세 정보입니다. |
| `top_cast` | 주요 출연진에 대한 정보입니다. |
| `details_release_date` | 미디어 콘텐츠의 공식 개봉일입니다. |
| `details_countries_of_origin` | 미디어가 제작된 국가입니다. |
| `details_official_site` | 미디어 콘텐츠의 공식 웹사이트 링크입니다. |
| `details_language` | 미디어 콘텐츠의 주요 언어입니다. |
| `details_also_known_as` | 미디어 콘텐츠의 대체 제목 또는 다른 명칭입니다. |
| `details_filming_locations` | 미디어가 촬영된 장소입니다. |
| `details_production_companies` | 미디어 제작에 참여한 회사입니다. |
| `specs_color` | 미디어의 색상 사양에 대한 정보입니다. |
| `specs_sound_mix` | 미디어의 사운드 믹스 사양에 대한 상세 정보입니다. |
| `specs_aspect_ratio` | 미디어의 화면비(Aspect Ratio) 사양입니다. |
| `url` | 미디어의 IMDB 페이지로 연결되는 URL 링크입니다. |
| `comment` | 미디어에 대한 사용자 댓글 또는 리뷰입니다. |
| `media_type` | 미디어 콘텐츠의 유형(예: 영화 또는 TV 쇼)입니다. |
| `imdb_rating` | IMDB에서의 미디어 평균 평점입니다. |
| `poster_url` | 미디어의 공식 포스터 이미지 URL입니다. |
| `imdb_rating_count` | IMDB에서 미디어가 받은 평점 수입니다. |
| `awards` | 미디어가 수상하거나 후보에 오른 पुरस्कार 정보입니다. |
| `critics_review_count` | 해당 미디어에 대해 उपलब्ध한 평론가 리뷰 수입니다. |
| `review_count` | 미디어에 대한 전체 사용자 리뷰 수입니다. |
| `review_rating` | 사용자 리뷰의 평균 평점입니다. |
| `featured_review` | 미디어에 대해 강조되거나 주목할 만한 리뷰입니다. |
| `storyline` | 미디어의 상세한 줄거리 또는 스토리 설명입니다. |
| `boxoffice_budget` | 미디어 콘텐츠의 제작 예산입니다. |
| `id` | 데이터셋 내 미디어 콘텐츠의 고유 식별자입니다. |


## Use Cases

### 1. Film Industry Analysis
시간 경과에 따른 영화 장르 및 인기도 추세를 분석하여 제작 전략과 마케팅 활동에 활용합니다.

### 2. Market Research
데이터셋을 사용하여 미디어 소비에서의 소비자 선호도와 행동을 연구하고, 타겟 광고에 활용합니다.

### 3. Academic Research
영화 제작, 개봉 상세 정보, 관객 반응에 대한 포괄적인 데이터를 제공하여 미디어 연구를 지원합니다.



## File Formats

데이터셋은 여러 형식으로 제공됩니다:
- **CSV** (이 샘플에 포함)
- **JSON**
- **NDJSON**
- **JSON Lines**
- **Parquet**
- **Compressed (.gz)**
